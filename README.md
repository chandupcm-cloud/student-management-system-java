Student Management System (Java) is a console-based application developed using Core Java that allows users to efficiently manage student records through a menu-driven interface. The system supports basic CRUD operations such as adding, viewing, searching, and deleting student details. It demonstrates key programming concepts including Object-Oriented Programming (OOP), encapsulation, and the use of Java Collections (ArrayList) for data storage. The project is structured into separate classes (Student, StudentService, and Main) to maintain clear separation of concerns and improve code organization. This application serves as a strong foundation for understanding core Java concepts and can be further enhanced by integrating a database or converting it into a Spring Boot REST API.
