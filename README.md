Student Management System (Java):
Overview:
        > This project is a console-based Student Management System developed using Core Java.
        > It allows users to manage student records efficiently through a menu-driven interface.
        > The application demonstrates fundamental programming concepts such as Object-Oriented Programming (OOP), Collections, and basic CRUD operations.
Features:
        > Add Student
        > View All Students
        > Search Student by ID
        > Delete Student
        > User-friendly menu-driven interface
Technologies Used:
        > Java
        > OOP Concepts (Encapsulation, Classes, Methods)
        > Array List (Java Collections Framework)
Project Structure:
src/
 └── project/
     ├── Main.java
     ├── Student.java
     ├── StudentService.java
How to Run?
      > Clone or download the repository
      > Open in Eclipse / VS Code
      > Run Main.java
      > Use the menu options to perform operations
Key Concepts Covered:
      > Object-Oriented Programming
      > Constructor & Method Usage
      > Collections (ArrayList)
      > Looping & Conditional Statements
      > Separation of Concerns (Model, Service, Main)
Future Enhancements:
      > Update Student Details
      > Use HashMap for faster search
      > Database integration (MySQL)
      > Convert into Spring Boot REST API
